Gery's First Webpage




Üdvözöllek az első HTML oldalamon. Ezt a weboldalt gyakorlás céljából hoztam létre, hogy kipróbálhassam a különböző HTML elemek használatát, gyakorolhassam a Bootstrap a JavaScrip és a CSS nyújtotta lehetőségeket. Nyugodtan próbálj is végig mindent, és néha nézz vissza az oldalra, hogyan fejlődik. Ezt leírom angolul is, hogy több legyen a szöveg: Welcome to my first HTML page. I created this website for practice so that I could try out the different HTML elements, practice Bootstrap with JavaScript and CSS. Feel free to try it all through and sometimes look back at how it develops.
